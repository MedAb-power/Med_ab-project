# UltimMC Setup

This repository contains the files for UltimMC, a custom Minecraft launcher setup.

## 📦 Features
- Easy Minecraft launching
- Supports multiple Minecraft versions
- Simple and lightweight
- Custom instances support
- Portable launcher files included

## 🚀 How to Use
1. Download or clone this repository
2. Open the `UltimMC.exe` file
3. Log in with your Minecraft account
4. Create or launch an instance
5. Start playing

## 🛠 Requirements
- Windows PC
- Java installed
- Internet connection for downloading game files

## 📁 Repository Contents
- Launcher executable
- Required DLL files
- Configuration files
- Minecraft launcher resources
